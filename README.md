# swiggy-sales-dashboard-excel
This project focuses on analyzing Swiggy sales data using Microsoft Excel to uncover meaningful insights related to sales performance, customer behavior, and regional trends. An interactive dashboard was developed to present data in a clear, visually appealing, and decision-friendly format.
🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Transformation
Dashboard Design
⚙️ Key Features & Implementation
Designed KPI metrics including Total Sales, Orders, and Ratings
Performed time-based analysis (Monthly, Weekly, Daily trends)
Identified Top 5 cities by sales performance
Created Veg vs Non-Veg analysis using a donut chart
Built a Quarterly performance summary table
Implemented Map visualization to show sales by state
Added interactive filters (Month, State, Restaurant)
Developed navigation buttons for smooth user experience
Applied Financial Year calculation logic using Excel formulas
📊 Dashboard Highlights
Interactive and dynamic filtering system
Clean and intuitive KPI cards
Visual trend analysis through charts
Geographic insights with map visualization
User-friendly and structured layout
🚀 Outcome

This dashboard provides valuable insights into sales trends, customer preferences, and regional performance, helping businesses make data-driven decisions effectively.
